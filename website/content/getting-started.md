# Getting Started

## Python

### Install the package

```bash
pip install egobox
```

## Rust

### Use the crates

```toml
[dependencies]
egobox-doe = { version = "0.x.y" }
egobox-gp = { version = "0.x.y" }
egobox-moe = { version = "0.x.y" }
egobox-ego = { version = "0.x.y" }
```
